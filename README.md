# tsconfig-react

TypeScript config preset for React.js projects.

[![CI](https://github.com/phuctm97/tsconfig-react/actions/workflows/ci.yml/badge.svg)](https://github.com/phuctm97/tsconfig-react/actions/workflows/ci.yml)

## Usage

### Install

<details>
  <summary>NPM</summary>

```shell
npm i -D tsconfig-react
```

</details>

<details>
  <summary>Yarn</summary>

```shell
yarn add -D tsconfig-react
```

</details>

### Use in `tsconfig.json`

```json
{
  "extends": "tsconfig-react",
  "exclude": ["node_modules"]
}
```
